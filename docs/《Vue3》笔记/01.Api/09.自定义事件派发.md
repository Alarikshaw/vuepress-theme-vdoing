### `emits`

`vue-next`多了一个派发事件的选项`emits`

在使用`emits`派发事件时，需要把事件名放入选项当中

```js
export default {
	emits: ['onClick']
}
```

如果派发的事件是一个原生`javaScript`事件，且没有放入`emits`里面，则该事件会被触发两次

### `v-model`

`vue-next`当中的`v-model`借助的属性是`modelValue`；借助的事件是`update:modelValue`,且移掉`sync`
